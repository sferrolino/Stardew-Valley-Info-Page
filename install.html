Installing Frameworks: Laravel & VueJS




Before Installing Frameworks

Install composer, a dependency management tool for php to help set up our larevel application.

Update the package manager

sudo apt update

Run the following to install dependencies for installing composer

sudo apt install curl php-cli php-mbstring git unzip

In your home directory, download composer installer using:

curl -sS https://getcomposer.org/installer -o composer-setup.php

Install composer globally

sudo php composer-setup.php --install-dir=/usr/local/bin --filename=composer

If there are no errors, you should now have composer installed.





Backend Framework

We will install a Laravel/VueJS application on a server with a LAMP stack and on Ubuntu 18.04.

Install some dependencies for laravel

sudo apt install php-xml php-bcmath

Now, we can create a new laravel app using:

composer create-project --prefer-dist laravel/laravel project_directory

Your new laravel project directory should now be created!





Frontend Framework

We will use scaffodling provided by Larave to install Vue

Inside your laravel project, run the following to install laravel/ui package

composer require laravel/ui

Now, install the Vue scaffolding with:

php artisan vue

php artisan ui vue--auth

Now, laravel's package dependencies will be updated to include vue. Install these dependencies with:

npm install

Finally, compile your assets using

npm run dev

Congratulations! You are done creating a laravel/vue application!
